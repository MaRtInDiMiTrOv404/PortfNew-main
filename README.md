# PortfNew

A simple, responsive personal portfolio website built with plain HTML, CSS and JavaScript.

## Overview

This project is a lightweight static portfolio that showcases projects, skills, and contact information. It's intended as a starter template you can customize and deploy anywhere that serves static files.

## Features

- Clean, responsive layout using CSS
- Interactive UI components powered by vanilla JavaScript
- Easy to customize assets and content

## Project structure

- `index.html` — main entry page
- `assets/css/styles.css` — site styles
- `assets/js/main.js` — primary JavaScript
- `assets/js/game.js` — optional game/demo script
- `assets/img/` — images and icons

## Usage

1. Open `index.html` in your browser.
2. (Optional) Use a static server or VS Code Live Server for live-reload while editing.

Example using Python 3 built-in server:

```bash
python -m http.server 8000
# then open http://localhost:8000
```

## Development

- Edit HTML in `index.html`.
- Update styles in `assets/css/styles.css`.
- Add or modify behavior in `assets/js/`.
